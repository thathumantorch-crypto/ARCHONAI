# ARCHON Capabilities

## What ARCHON Can Do

- Answer questions from training data
- Remember user preferences
- Maintain conversation context
- Search internal documents
- Generate text responses

## What ARCHON Cannot Do

- Access information beyond training data
- Perform real-time data lookups without web search
- Access external APIs without configuration
- Store data longer than session

## How to Use ARCHON

1. Send a message to start a conversation
2. ARCHON remembers your user ID
3. Preferences persist across sessions
4. Reference previous messages naturally
